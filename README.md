这是个 [Next.js](https://nextjs.org/) + [Tauri](https://tauri.app/) 的项目。

## 初始化项目

1. 安装依赖:

```bash
yarn
```
---

2. 开发server:

> 整体应用

```bash
yarn run dev
```

> 仅 next (前端)

```bash
yarn run dev::next
```

打开 [http://localhost:3000](http://localhost:3000) 可以看到结果
