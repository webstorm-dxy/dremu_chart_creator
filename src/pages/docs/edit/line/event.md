| 事件名  |                     事件描述                     |
| :-----: | :----------------------------------------------: |
|  Notes  |                   线下的Note。                   |
|  Dots   | 绘制线的Dots，线的起始结束，形状均有此事件决定。 |
|  Moves  |                 改变线的坐标基点                 |
| Alphas  |             透明度事件，无事件时为1              |
| Rotates |             线围绕坐标基点旋转的事件             |
| Timings |                   控制线的流速                   |
